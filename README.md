# 📈 StockPulse: End-to-End Market Forecasting Pipeline

StockPulse is a production-grade machine learning application that predicts the next day's closing price for any publicly traded stock. It leverages a **Long Short-Term Memory (LSTM)** neural network and integrates real-time financial data via the Yahoo Finance API.

## 🚀 Key Features
* **Real-Time Data Ingestion:** Uses `yfinance` to fetch live market data including Open, High, Low, Close, and Volume.
* **Technical Indicator Suite:** Features a robust engineering pipeline that calculates 9 technical indicators (Moving Averages, RSI, Volatility, etc.) for every prediction.
* **Deep Learning Engine:** Powered by an LSTM architecture designed to capture long-term dependencies in time-series data.
* **Web Interface:** Deployed via Gradio for a seamless, user-friendly forecasting experience.

## 🧠 Model Architecture & Performance
The model was built using **TensorFlow/Keras** with the following specifications:
* **Architecture:** Multi-layered LSTM with Dropout layers to prevent overfitting.
* **Sequence Length:** 60 days (the model "looks back" at the previous 2 months of data to predict tomorrow).
* **Optimization:** Adam optimizer with Mean Squared Error (MSE) loss.
* **Accuracy Metrics:**
    * **MAPE (Mean Absolute Percentage Error):** 5.79%
    * **MAE (Mean Absolute Error):** ₹224.62 (on TCS test data)
    * **RMSE (Root Mean Squared Error):** ₹265.45

## 🛠️ Technical Indicators (Feature Engineering)
The model processes 9 distinct features to understand market momentum and trends:
1.  **Close Price:** The primary target.
2.  **MA7 & MA21:** Short and medium-term moving averages.
3.  **MA50:** Long-term trend indicator.
4.  **RSI (Relative Strength Index):** To identify overbought or oversold conditions.
5.  **Volatility:** 7-day rolling standard deviation.
6.  **Price Range:** Daily High-Low spread.
7.  **Daily Return:** Percentage change from the previous day.
8.  **Volume MA7:** 7-day average of trading volume to detect institutional activity.

## 💻 Installation & Local Usage
To run the predictor locally, clone the repository and install the dependencies:

```bash
git clone https://github.com/yourusername/stockpulse.git
cd stockpulse
pip install -r requirements.txt
python app.py
```
